# Saeed — Data Analyst Portfolio

SQL · Power BI · ETL · DAX · PostgreSQL · MySQL

---

This portfolio contains six end-to-end data analysis projects covering relational database engineering, business intelligence dashboards, and operational reporting. The projects span logistics, hospitality, aviation, marketing, and warehouse management — all built as training case studies on realistic industry datasets.

---

## SQL Projects

Three case studies using MySQL and PostgreSQL, covering multi-table joins, window functions, CTEs, subqueries, and reusable views.

| Project | Dataset | Focus |
|---|---|---|
| [Northwind Foods — Post-Merger Analysis](https://github.com/saeedullah-analyst/northwind-data-analysis) | Global B2B food distributor | Supply chain, inventory capital, logistics SLA, geographic revenue |
| [Unique Films Rental — Operational Audit](https://github.com/saeedullah-analyst/unique-films-rental-analysis) | Video rental company (Sakila) | Schema audit, financial aggregations, window functions, correlated subqueries |
| [Hotel Reservation Analysis](https://github.com/saeedullah-analyst/hotel-reservation-analysis) | Hotel booking ledger | Front-desk queries, cancellation patterns, lead-time analysis |

## Power BI Dashboards

Three dashboards built in Power BI Desktop using Power Query for ETL, star schema data modeling, and DAX for calculated measures.

| Project | Dataset | Focus |
|---|---|---|
| [Bearbus Warehouse Dashboard](https://github.com/saeedullah-analyst/warehouse-logistics-dashboard) | 258K warehouse transactions | Processing delays, slot optimisation, picking accuracy, weekend backlogs |
| [TechFuel Social Media KPI Tracker](https://github.com/saeedullah-analyst/marketing-kpi-dashboard) | Multi-platform campaign logs | Platform engagement rates, content theme ranking, follower target tracking |
| [LAX Flight Punctuality Analysis](https://github.com/saeedullah-analyst/lax-flight-punctuality-analysis) | 1.26M flight records (2015–2017) | Arrival vs. departure delay asymmetry, carrier ranking, hourly/seasonal patterns |
